Dungeon Diving

As I mentioned this projects purpose was to teach us communication skills as well as how to work with other people on a real project.

What I contributed to this project:

  • Enemy AI
  • Special player ability
  • Built the combat system(based off of pokemon)
  • Added potion functionality
  • Created chests that are found throughout the game
  • Add button functionality to various things

Through this project I was able to learn the importance of communication, especially working with version control (Tortoise SVN), as well as the development process.
